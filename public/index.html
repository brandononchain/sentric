Protocol

Signal. Score. Serve.

Three phases, zero human intervention. Raw wallet activity becomes tradeable intelligence for autonomous agents.

01

Ingest

Subscribes to 512 KOL wallets via Helius WebSocket. Every swap on Jupiter, Raydium, and Meteora captured within 400ms of block confirmation.

02

Score

Five on-chain metrics: position size, hold duration, historical PnL, rug avoidance, multi-wallet consensus. Output: 0-100 conviction score.

03

Serve

Scored signals exposed via x402-gated HTTP endpoints. $0.001 per signal in USDC. No API keys. No subscriptions. Sign and fetch.

Developer Experience

One request. One micropayment.

consume-signal.ts
// No API key. No account. Just USDC.

import { SolanaAgentKit } from "solana-agent-kit";
import { SentryPlugin } from "@sentry/agent-kit";

const agent = new SolanaAgentKit(wallet, rpc)
  .use(SentryPlugin);

const signals = await agent.getSentrySignals({
  minConviction: 80,
  maxAge: 60,
});

for (const s of signals) {
  if (s.score > 90) {
    await agent.trade(s.tokenMint, s.size);
  }
}
