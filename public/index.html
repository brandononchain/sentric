Protocol

Signal. Score. Serve.

Three phases, zero human intervention. Sentry converts raw wallet activity into tradeable intelligence for autonomous agents.

01

Ingest

Subscribes to 512 known KOL wallets via Helius WebSocket streams. Every swap on Jupiter, Raydium, and Meteora is captured in under 400ms from block confirmation.

02

Score

Five on-chain metrics: position size relative to portfolio, hold duration, historical PnL, rug avoidance rate, and multi-wallet consensus. Output: a 0-100 conviction score.

03

Serve

Scored signals exposed via x402-gated HTTP endpoints. Any agent pays $0.001 per signal in USDC. No API keys. No subscriptions. Just sign and fetch.

Developer Experience

One request. One micropayment. One signal.

consume-signal.ts 
// Agent consumes Sentry signals via x402
// No API key. No account. Just USDC.

import { SolanaAgentKit } from "solana-agent-kit";
import { SentryPlugin } from "@sentry/agent-kit";

const agent = new SolanaAgentKit(wallet, rpc)
  .use(SentryPlugin);

const signals = await agent.getSentrySignals({
  minConviction: 80,
  maxAge: 60,
});

for (const s of signals) {
  if (s.score > 90) {
    await agent.trade(s.tokenMint, s.size);
  }
}
